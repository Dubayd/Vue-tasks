# Vue-tasks

# Vue To-Do Lijst

Dit is een simpele app die ik gebouwd heb met behulp van Vue 3.

## Features

- Voeg nieuwe taken aan je to-do-lijst.
- Zie al je toegevoegde taken in ene lijst.
- Verwijder taken simpel door op 'X' te klikken.
